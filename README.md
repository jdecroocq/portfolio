This repository contains the source code and some media assets of my personal portfolio website.<br><br>

The `main` branch is the production site, deployed at [jdecroocq.vercel.app](https://jdecroocq.vercel.app/).\
The `dev` branch is used for experimenting and testing new features, deployed at [jdecroocq-dev.vercel.app](https://jdecroocq-dev.vercel.app/).\
The `content` branch is dedicated to updating non-code content (texts, media, data), deployed at [jdecroocq-content.vercel.app](https://jdecroocq-content.vercel.app/).<br><br>
